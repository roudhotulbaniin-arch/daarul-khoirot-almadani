README - Repositori Website Daarul
Khoirot Almadani
Dokumen ini memberikan panduan dan informasi mengenai struktur serta konten dari repositori
website resmi Lembaga Pendidikan Daarul Khoirot Almadani.
Profil Lembaga
Daarul Khoirot Almadani adalah lembaga pendidikan Islam yang berdedikasi untuk
membentuk generasi Qur'ani yang berakhlak mulia. Lembaga ini berfokus pada integrasi
pendidikan agama yang mendalam dengan pembentukan karakter santri.
Unit Pendidikan
Website ini mencakup informasi mengenai tiga unit utama pendidikan kami:
Unit Pendidikan Deskripsi Program
TPQ Roudhotul Baniin Pendidikan dasar baca tulis Al-Qur'an
dengan fokus pada Tahsin dan Tajwid untuk
anak usia dini.

MDT Roudhotul Baniin Madrasah Diniyah Takmiliyah yang fokus
pada pendalaman Fiqih, Aqidah, dan
Bahasa Arab.

Ponpes Daarul Khoirot Almadani Pondok Pesantren dengan program utama
Tahfidzul Qur'an dan kajian Kitab Kuning
secara intensif.

Fitur Utama Website
● Beranda: Informasi umum dan visi-misi lembaga.
● Galeri: Dokumentasi kegiatan santri dan fasilitas lembaga.
● Pendaftaran: Informasi dan formulir pendaftaran santri baru.
● Tadzkiroh: Kumpulan materi dakwah dan tausiyah.
● Info Islam: Artikel dan wawasan keislaman.
● Kontak: Layanan tanya jawab cepat melalui integrasi WhatsApp.
